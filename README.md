## Running locally

Git clone

```bash
  git clone https://link-para-o-projeto
```

Open the project

```bash
  cd my-project
```

Install all the dependencies (if you have some problems just type "--force" at the end)

```bash
  npm install
```

Run the node server

```bash
  npm run start
```
